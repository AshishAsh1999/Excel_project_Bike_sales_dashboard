# Bike Sales Dashboard (Excel)

## Project Overview
This project presents an interactive Excel dashboard designed to analyze bike purchasing behavior across different customer segments. The dashboard enables users to explore how demographic, income, education, and commute related factors influence bike purchases.

The analysis is built using Excel pivot tables, pivot charts, slicers, and standard data preparation techniques.

---

## Dashboard Features
- Interactive slicers for marital status, region, and education
- Average income analysis by gender and purchase decision
- Customer age bracket comparison against bike purchase behavior
- Commute distance analysis across purchase categories
- Structured and business oriented dashboard layout

## Key Insights
- Middle aged customers demonstrate the highest likelihood of purchasing bikes
- Customers with higher average income are more likely to make a purchase
- Shorter commute distances correlate with higher purchase rates
- Education level and region influence purchasing behavior
